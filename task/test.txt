Тестовое задание.

Сверстать адаптивную форму входа на сайт.
Для выполнения работы нужно клонировать репозиторий git clone https://Feschenko@bitbucket.org/Feschenko/test.git
Выполненное задание залить в отдельную ветку Имя_Фамилия

По возможности использовать и реализовать:
- методологию БЭМ;
- проект собрать при помощи Gulp;
- клиентскую валидацию полей (поля не должны быть пустыми, имя может содержать буквы, цифры, пробел, тире, апостроф);
- Если логин и пароль прошли валидацию запомнить их и при следующем обновлении показать экран приветствия.

На что буду обращать внимание:
- соответствие макету;
- семантичность разметки, именование классов;
- адаптивность (форма должна быть полностью адаптивной до 320px).

Библиотеки, препроцессоры на рассмотрение исполнителя.
